# Obsidian 同步保险库插件

## 简介

这是一个为 Obsidian 设计的插件,旨在提供一种简单而高效的方式来同步您的 Obsidian 保险库。用来将 Obsidian 的笔记同步到远端云盘。

## 特性

- 安全的文件同步: 使用加密技术确保您的数据在传输过程中的安全。
- 自动同步: 在加密模式中，自动将本地修改的文件同步到远端。
- 选择性同步: 允许用户选择要同步的特定文件夹或文件。
- 跨平台支持: 适用于 Windows、MacOS、Linux、iOS、iPadOS和Android。

## 安装

1. 打开 Obsidian 设置
2. 导航到 "第三方插件"
3. 确保 "安全模式" 已关闭
4. 下载本插件，解压后将文件夹重命名为 `obsidian-sync-vault`
5. 将文件夹放置在 Obsidian 的插件目录下，通常位于 `VaultFolder/.obsidian/plugins`
6. 安装完成后,启用插件

## 使用方法

1. 在 Obsidian 中,打开插件设置
2. 点击授权获取授权码
3. 点击激活，输入激活码
4. 激活成功后，再次打开插件，即可开始使用

详细的使用说明请参考 [使用文档](https://kqiu.top/sync-obsidian-with-baidu-cloud-disk/)。

## 配置

插件提供了以下配置选项:

- 是否在加载时自动同步，建议开启
- 是否加密，建议开启
- 加密密码，请牢记
