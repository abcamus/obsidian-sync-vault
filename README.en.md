# Obsidian Sync Vault Plugin

## Introduction

This is a plugin designed for Obsidian, aimed at providing a simple and efficient way to synchronize your Obsidian vault. It's used to sync Obsidian notes to a remote cloud drive.

## Features

- Secure file synchronization: Uses encryption technology to ensure the safety of your data during transmission.
- Automatic synchronization: In encryption mode, automatically syncs locally modified files to the remote.
- Selective synchronization: Allows users to choose specific folders or files to sync.
- Cross-platform support: Available for Windows, MacOS, Linux, iOS, iPadOS, and Android.

## Installation

1. Open Obsidian settings
2. Navigate to "Third-party plugins"
3. Make sure "Safe mode" is turned off
4. Download this plugin, unzip it and rename the folder to `obsidian-sync-vault`
5. Place the folder in Obsidian's plugin directory, typically located at `VaultFolder/.obsidian/plugins`
6. After installation, enable the plugin

## Usage

1. Open the plugin settings in Obsidian
2. Click on authorize to get the authorization code
3. Click activate and enter the activation code
4. After successful activation, open the plugin again to start using it

For detailed instructions, please refer to the [User Guide](https://kqiu.top/sync-obsidian-with-baidu-cloud-disk/).

## Configuration

The plugin provides the following configuration options:

- Whether to automatically sync on loading, recommended to enable
- Whether to encrypt, recommended to enable
- Encryption password, please remember it well
